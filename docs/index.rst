Introduction
============

:mod:`ndfilters` is an experimental library designed to parallelize the filters in
:mod:`scipy.ndimage` using :mod:`numba`.


Installation
============
:mod"`ndfilters` is published on PyPI and can be installed using::

    pip install ndfilters
